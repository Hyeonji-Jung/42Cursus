Block

공통된 html 코드 부분을 base.html이라는 별도의 파일로 작성하여, 다른 페이지(자식)들이 상속(extends)받게 할 수 있습니다.

base.html 파일의 작성과 상속을 위해 각 파일은 다음과 같은 형식으로 작성됩니다.

base.html {% verbatim %}
  • {% block contents %}
  • 다른 페이지의 html 코드가 들어갈 장소
  • {% endblock %}
{% endverbatim %} 다른 html 파일들 {% verbatim %}
  • {% extends "base.html" %}
  • {% block contents %}
  • 해당 페이지에서 사용하는 html코드
  • {% endblock %}
{% endverbatim %}

for

배열의 각 원소에 대하여 반복(루프)하는 것을 말합니다.

간단한 예로, student_list에 들어 있는 선수의 목록을 출력하기 위해 아래의 예제처럼 사용할 수 있습니다.

{% verbatim %}
  • {% for student in student_list %}
  • {{ student.name }}
  • {% endfor %}
{% endverbatim %}

if / else

변수가 true이면 블록의 컨텐츠를 표시하는 기능으로, if 태그 내에 템플릿 필터 및 각종 연산자를 사용할 수 있습니다.

{% verbatim %}
  • {% if student_list %}
  • 총 학생 수 : {{ student_list|length }}
  • {% else %}
  • 학생이 없습니다.
  • {% endif %}
{% endverbatim %}
